# CloudWhalesBlogCore
AspNetCore5.0-项目搭建-从办公自动化开始
是将表格图表数据加水印导出的小项目，一开始时客户进行人工复制贴图，完成一个143张图的文档需要4-6小时，使用工具后最大的文档尚未超过3分钟。

除功能的实现之外，更多的是训练项目层次构建逻辑，摆脱以往构建项目层次感较弱的现状
